A simple telegram bot to notify users on new weather warnings. The bot will send
messages to a telegram user or group once the user subscribed to new weather updates. In addition,
the user can query the current weather.

How to run this weather bot?
1. Create a Telegram bot from BotFather, and get a bot token.
2. Modify <token> as your token(get by step 1) in BotConfig.java
3. Import following jar, and run this code.
